# Eyenalysis
A Noninvasive Disease Detection System that uses multiple binary classification based neural networks in order to detect over 150 different eye diseases.

## Requires:
  - Python 3.6
  - Tensorflow 2.0>=
  - OpenCV
  
## Instructions:
1) Clone the repository
2) Replace the filepaths in the respective model python file
3) Run and use the inference script.
